1. 수업에서 나온 질문에 대한 답변을
   → **다음 스터디 전까지 GitHub에 각자 업로드합니다.**

   → **스터디 당일, GitHub 이슈에 정리해서 기록합니다.**

   → **Slack `#android-7기-잡담-질답`에 공유합니다.**
